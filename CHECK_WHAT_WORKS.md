# Детальная проверка работы плагина

## Как проверить

### 1. Запустить плагин
```bash
./gradlew runIde
```

Дождаться загрузки IDE (PhpStorm).

### 2. Создать или открыть ASP файл

В открывшейся IDE:
1. File → New → ASP File (если есть) или создать test.asp
2. Или открыть существующий test-example.asp из проекта

## Что должно работать (чеклист)

### ✅ Базовое распознавание файла
- [ ] Файл .asp открывается в редакторе
- [ ] Внизу справа показывается тип файла "ASP"
- [ ] В структуре проекта у файла .asp своя иконка

### ✅ HTML Подсветка синтаксиса
Вставьте этот код в .asp файл:
```html
<html>
<head>
    <title>Test</title>
    <style>
        body { color: red; }
    </style>
</head>
<body>
    <h1>Hello</h1>
    <p>World</p>
</body>
</html>
```

Проверьте:
- [ ] HTML теги `<html>`, `<head>`, `<body>` подсвечиваются одним цветом
- [ ] Атрибуты подсвечиваются другим цветом
- [ ] Текст внутри тегов обычным цветом
- [ ] CSS внутри `<style>` имеет CSS подсветку

### ✅ Автодополнение
- [ ] Начните печатать `<di` и нажмите Ctrl+Space (Cmd+Space на Mac)
- [ ] Должны появиться подсказки: `<div>`, `<dialog>`, etc.
- [ ] Выберите `<div>` - должен автоматически добавиться `</div>`
- [ ] Внутри тега начните печатать атрибут `cl` и нажмите Ctrl+Space
- [ ] Должна появиться подсказка `class`

### ✅ Валидация HTML
Создайте невалидный HTML:
```html
<div>
    <p>Text without closing tag
</div>
```

Проверьте:
- [ ] Должна появиться ошибка/предупреждение о незакрытом `<p>`
- [ ] Ошибка подсвечивается красным/оранжевым

### ✅ Навигация по тегам
```html
<div class="container">
    <div class="inner">
        <p>Text</p>
    </div>
</div>
```

Проверьте:
- [ ] Кликните курсором на открывающий `<div>` - должен подсветиться соответствующий `</div>`
- [ ] То же для любого другого тега

### ✅ Code Folding (сворачивание кода)
```html
<div>
    <p>Line 1</p>
    <p>Line 2</p>
    <p>Line 3</p>
</div>
```

Проверьте:
- [ ] Слева от `<div>` должна появиться стрелка вниз
- [ ] Клик по ней сворачивает содержимое `<div>`

### ✅ Форматирование кода
Создайте неформатированный HTML:
```html
<html><head><title>Test</title></head><body><h1>Hello</h1></body></html>
```

Проверьте:
- [ ] Нажмите Ctrl+Alt+L (Cmd+Option+L на Mac)
- [ ] Код должен отформатироваться с правильными отступами

### ✅ Структура документа
- [ ] Откройте Structure View (Alt+7 или Cmd+7)
- [ ] Должна показываться структура HTML тегов

## Что НЕ должно работать (это нормально)

### ❌ ASP/VBScript код
```asp
<%
Dim name
name = "John"
Response.Write(name)
%>
```

Это будет отображаться как обычный текст или HTML комментарий.
**Это нормально** - мы пока не парсим VBScript.

## Если что-то не работает

### Проверьте логи:
```bash
tail -f build/idea-sandbox/PS-2024.3.1.1/log/idea.log
```

### Проверьте, что плагин установлен:
1. Settings → Plugins → Installed
2. Должен быть "ASP Classic Language Support"
3. Если нет - перезапустите IDE

### Переустановите плагин:
```bash
./gradlew clean build runIde
```

## Сообщите, что не работает

Если что-то из HTML функциональности НЕ работает, сообщите:
1. Что именно не работает (из чеклиста выше)
2. Что показывается вместо ожидаемого
3. Есть ли ошибки в логах

Тогда мы сможем точечно исправить проблему!
